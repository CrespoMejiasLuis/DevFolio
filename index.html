Projects

La Llama de Sant Antoni

A team-developed game centered around the traditional Sant Antoni festival at Vilanova D'Alcoleu. Awarded with best narrative,best cultural adaptation and best cohesive art. My main focus was the narrative direction and cultural investigation. I also helped with the game design programming and art.

Keywords: Group Project · Unity · 2D Game · Narrative

PinguinEscape

A stealth game where you control a penguin escaping from AI-driven soldiers. My main focus was the level Design, but I also participated in the game AI and game mechanics.

Keywords: Unity · Team Game ·A* Pathfinding · Machine States

GravityIsRelative

An endless runner where the core mechanic revolves around flipping gravity. Players toggle between normal and inverted physics to dodge obstacles and progress. Fully developed by me.

Keywords: Endless Runner · Gravity · Physics · Platformer · Unity

Carcanoid

A car-themed Arkanoid (brick-breaker) game built with the Phaser JS framework. My main focus was the core mechanics programming and the game feel.

Keywords: Phaser · JavaScript · Web Game · Arkanoid · · HTML5

UjiBan

A Sokoban-style puzzle game for Android featuring a built-in level designer and a database backend for saving custom puzzles. School project where my main focus was the gameplay mechanics programming and the level editor.

Keywords: Android · Puzzle · Sokoban · Level Editor · Mobile · Database

WaterDrop

A physics-based platformer demo where a water-drop character navigates through levels. Highlights physics interactions and character movement.

Keywords: Platformer · 2D

MarioKart 3D Model

A detailed 3D model of a Mario Kart vehicle, created in 3ds Max.

Keywords: 3ds Max · 3D Modeling · Texturing · Rendering

FlatIron Building

An accurate architectural recreation of New York's iconic Flatiron Building, modeled using AutoCAD and rendered in Blender for a high-fidelity visualization.

Keywords: AutoCAD · Blender · Architecture · 3D Modeling · Visualization
